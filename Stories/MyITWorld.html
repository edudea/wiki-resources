Ziele

Mit den Stories soll eine mögliche IT Umgebung für z.B. Familien, KMU und Vereine entwickelt werden, die im wesentlichen folgenden Ansprüchen genügt:

  • Selbstbestimmter Umgang mit den Daten, Verwendung von Open Source Software
  • Zukunftsorientiert
  • Verwendung von Standard Hardware (Desktop, Notebook, Tablet und Smartphone)
  • Berücksichtigung der Kommunikation im Team und mit externen
  • Kostengünstig
  • Mit geringen IT Kenntnissen nachvollziehbar
  • Keine Fremdsprachenkenntnisse erforderlich
  • Erweiterbar für spezifische Anwendungen

Die hier beschriebene IT Welt soll eine Alternativen zur Nutzung der Datenkraken aufzeigen, deren Geschäftsmodell die kommerzielle Nutzung unserer Daten ist - dies ohne wesentliche Einbussen des Benutzungskomforts oder der Kompatibilität.

Folgende Überlegungen:

  • Keine eigenen Server, da dafür zu hohe IT Kenntnisse erforderlich sind
  • Nutzung von Cloud Diensten als SaaS

Allgemeine Anforderungen

Um obengenannte Ziele für die IT Umgebung zu erreichen, werden folgenden Anforderungen aufgestellt:

  • Weitgehende Nutzung von Open Source Software
  • Nutzungsmöglichkeit auf verschieden Geräten wie Desktop, Notebook, Raspberry, Smartphone oder Tablet
  • Möglichst keine Installation von APPs, d.h. Nutzung über den Browser
  • Geschützter Zugriff, 2 Faktorauthentifizierung bei privaten oder personenbezogenen Daten, Passwörter und Schlüssel nicht beim Provider (Zero-Knowledge)
  • Daten werden nicht für Werbezwecke verwendet
  • Private oder personenbezogenen Daten können vom Provider möglichst nicht eingesehen werden
  • Kontrolle über alle vom Provider gespeicherten Daten (Umfang bekannt, änderbar, löschbar)
  • Anbieter sollte nicht dem US Cloud Act oder anderen staatlichen Zugangsmöglichkeiten unterliegen
  • Zugangsberechtigung
    • Adminfunktionen
    • Gruppen
    • Mitmachen (evtl. eingeschränkte Funktionalität) ohne permanente Anmeldung möglich
    • Unterstützt Account-Management System
